# java-lab

## 📖 Sobre o projeto / About the project
Este repositório é um **laboratório de estudos em Java**, criado para reunir conteúdos, exercícios e experiências práticas durante a jornada de aprendizado da linguagem. A ideia é explorar conceitos fundamentais e avançados, documentando cada etapa com exemplos e anotações.  
This repository is a **Java study lab**, created to gather content, exercises, and practical experiences throughout the learning journey with the language. The idea is to explore both fundamental and advanced concepts, documenting each step with examples and notes.

## 🎯 Objetivos / Goals
- Consolidar conhecimentos fundamentais de Java.  
- Experimentar recursos da linguagem e boas práticas de programação.  
- Criar exemplos práticos que possam servir como referência futura.  
- Registrar a evolução dos estudos e aprendizados.  
---  
- Consolidate fundamental knowledge of Java.  
- Experiment with language features and programming best practices.  
- Create practical examples that can serve as future references.  
- Record the evolution of studies and learnings.
